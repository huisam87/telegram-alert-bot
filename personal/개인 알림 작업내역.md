# 개인 알림 작업내역

## 개요
- 텔레그램으로 AI 업데이트 요약을 매일 09:00(KST) 받는 개인 알림 봇 구성
- 네이버/구글 뉴스 + 지정 유튜브 채널 RSS 수집 후 요약

## 환경/연결
- GitHub SSH 연결 및 gh 로그인 완료
- 텔레그램 봇 생성 및 chat ID 연결 완료

## 생성/수정된 주요 파일
- `C:\codex\personal\personal.md` : 요구사항/포맷/소스 정리 문서
- `C:\codex\personal\telegram_send.ps1` : 텔레그램 단순 전송 테스트
- `C:\codex\personal\.env.example` : 환경변수 예시
- `C:\codex\personal\telegram_send.md` : 테스트 스크립트 설명
- `C:\codex\personal\ai_news_config.json` : 키워드/도구/채널/옵션 설정
- `C:\codex\personal\ai_news_digest.py` : 뉴스/유튜브 수집 + 요약 생성
- `C:\codex\personal\ai_news_send.ps1` : 요약 생성 + 텔레그램 전송
- `C:\codex\personal\ai_news_send.md` : 실행/동작 설명

## 기능 요약
- 기본 도구: Gemini, ChatGPT, Claude
- 해외 인기 AI 툴 섹션 추가(Claude 아래 고정)
- 한국어 기사 우선, 해외 인기 AI 툴 섹션은 영어 기사 허용
- 링크는 말줄임 + HTML 링크로 전송
- 유튜브 채널 고정: jocoding, aidia-v5w, ScienceADAM, nomadcoders, codingapple
- 인물 기사: 김대식, 송길영, 김덕진

## 자동 실행
- 작업 스케줄러 등록: 매일 09:00 실행 (작업명: `AI_News_Digest`)
- 실행 명령: `powershell -ExecutionPolicy Bypass -File C:\codex\personal\ai_news_send.ps1`

## 사용 방법
- 테스트 실행:
  - `powershell -ExecutionPolicy Bypass -File C:\codex\personal\ai_news_send.ps1`
- 스케줄 확인:
  - `schtasks /Query /TN "AI_News_Digest" /V /FO LIST`
